# Wave Icons Pack

10 wave-themed SVG icons in **5 elements × 4 seasons** = 200 total SVGs.

## Elements & Seasons

| Element | 🌸 Spring | ☀️ Summer | 🍂 Fall | ❄️ Winter |
|---------|-----------|-----------|---------|-----------|
| 🌊 **Water** | Fresh teals | Deep ocean blues | Stormy slate | Icy blues |
| 🌍 **Earth** | Brown + kelly green + canary | Burnt brown + dark green + orange | Brown + orange + red | Light brown + gray + light blue |
| 💨 **Wind** | Soft pastel sky | Bright clear sky | Storm grays | Pale frost |
| 🔥 **Fire** | Warm peach | Intense red-orange | Deep amber | Muted cool reds |
| ✨ **Ether** | Light gray + mint | Stark black/white | Warm taupe | Icy white + silver |

## Icons

Line, Layered, Circle, Audio Waveform, Sun, Seamless Pattern, Tsunami, Minimal, Mountains, Spiral

## Quick Start

```
git clone https://github.com/LClay5/wave-icons-pack.git
```

Open `preview.html` to browse all themes.

## Build from source

```
node build.js
```

Edit `src/*.svg` (change shapes) or `themes/palettes.json` (change colors).
