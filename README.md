# Wave Icons Pack

10 wave-themed SVG icons in **5 elements × 4 seasons × day/night** = 400 SVGs.

| Element | 🌸 Spring | ☀️ Summer | 🍂 Fall | ❄️ Winter |
|---------|-----------|-----------|---------|-----------|
| 🌊 Water | Fresh teal | Deep ocean | Stormy slate | Icy blue |
| 🌍 Earth | Brown + kelly green + canary | Burnt brown + dark green + orange | Brown + orange + red | Light brown + gray + light blue |
| 💨 Wind | Pastel sky | Bright sky | Storm gray | Pale frost |
| 🔥 Fire | Warm peach | Red-orange | Deep amber | Cool muted red |
| ✨ Ether | Gray + mint | Black/white | Taupe | Icy white + silver |

Each season has a **☀️ Day** (custom palette) and **🌙 Night** (auto-darkened).

## Icons

Line, Layered, Circle, Audio Waveform, Sun, Seamless Pattern, Tsunami, Minimal, Mountains, Spiral

## Quick Start

```
git clone https://github.com/LClay5/wave-icons-pack.git
```

Open `preview.html` to browse all 40 theme combos.

## Build

Edit `src/*.svg` (shapes) or `themes/palettes.json` (day colors), then:

```
node build.js
```

Night variants are auto-generated by darkening day colors.
